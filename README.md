# Python图书借阅推荐系统 python155

## 项目概述

本项目是基于Python开发的图书借阅推荐系统，集成了django框架、MySQL数据库以及Vue.js前端技术。系统致力于为用户提供便捷的图书检索、借阅及信息管理服务。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：Vue.js

## 功能模块

### 核心功能

- 图书借阅
- 图书信息查阅
  - 最新图书
  - 最热图书
  - 推荐图书

### 管理功能

- 图书搜索
- 图书分类
- 图书详情查看
- 图书信息管理
- 分类管理
- 评论管理
- 标签管理

## 系统角色

- 普通用户：进行图书检索、借阅等操作
- 管理员：管理图书信息、分类、评论及标签等

## 运行环境

- Python环境
- MySQL数据库
- Vue.js运行环境

## 部署步骤

1. 配置Python环境
2. 安装相关依赖库
3. 配置MySQL数据库
4. 部署前端Vue.js
5. 运行后端Django服务
